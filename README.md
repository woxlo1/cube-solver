# 🧩 Rubik's Cube Solver

3×3 & 4×4 ルービックキューブの解法ガイドWebアプリです。  
GitHub Pagesで動く完全クライアントサイド実装（サーバー不要）。

## ✨ 機能

| 機能 | 詳細 |
|------|------|
| **3×3 対応** | IDA* アルゴリズムで最短解法を自動計算（最大22手） |
| **4×4 対応** | リダクション法（センター → エッジ → 3×3）ステップガイド |
| **手動入力** | パレットで色を選んでマスをクリック入力 |
| **カメラ入力** | スマホ/PCカメラで面を撮影して色を自動検出 |
| **ステップガイド** | 展開図リアルタイム更新 + 矢印アニメーション + 日本語説明 |
| **自動再生** | 1.4秒ごとに手順を自動進行 |

## 🚀 使い方

### ローカルで開く
```
index.html をブラウザで直接開くだけで動作します。
```

### GitHub Pages で公開する

1. GitHub でリポジトリを作成（例: `rubiks-solver`）
2. `index.html` と `README.md` をアップロード
3. リポジトリの **Settings → Pages → Source: Deploy from branch (main)**
4. `https://<ユーザー名>.github.io/rubiks-solver/` でアクセス可能

## 📖 操作手順

### 色の入力（6面分）
1. 上面(U) → 前面(F) → 右面(R) → 後面(B) → 左面(L) → 下面(D) の順に入力
2. **手動入力**：右パレットで色を選び、マスをクリックして色付け
3. **カメラ入力**：「📷 カメラ入力」タブ → ガイドボックスに面を合わせて撮影 → 修正して「この色を使用」

### 解法の確認
- 「解法を計算」ボタンで自動計算
- **← 戻る / 次へ →** でステップを手動移動
- **▶ 自動再生** で自動進行

## 🔡 記法（公式記法）

| 記号 | 意味 |
|------|------|
| `U` | 上面を時計回り90° |
| `U'` | 上面を反時計回り90° |
| `U2` | 上面を180° |
| `R / L / F / B / D` | 各面（同様の法則） |
| `Uw / Rw` | 4×4用：2層まとめて回す |
| `[STAGE n]` | 4×4の作業ステージ |

## 🔧 技術スタック

- **HTML / CSS / JavaScript** — フレームワーク不要、1ファイル完結
- **IDA* アルゴリズム** — 3×3の最適解探索
- **MediaDevices API** — カメラアクセス
- **Canvas API** — カメラ画像からの色サンプリング

## ⚠️ 注意事項

- カメラ入力は **HTTPS環境**（GitHub Pages含む）またはローカル(localhost)でのみ動作します
- 4×4の自動解法計算はステージガイド形式です（完全自動計算は現在非対応）
- 3×3のIDA*は深度が深い場合、計算に数秒かかることがあります
